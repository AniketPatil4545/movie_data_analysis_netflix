# 🎬 Movie Data Analysis with Python

## 📖 Overview

This project performs Exploratory Data Analysis (EDA) on a movie dataset using Python. The goal is to clean the dataset, analyze movie trends, and visualize insights related to popularity, genres, ratings, and release years.

The project demonstrates essential data analysis techniques using Pandas and creates visualizations to better understand the dataset.

---

## 🎯 Objectives

- Clean and preprocess the movie dataset.
- Analyze movie popularity and ratings.
- Explore genre distribution.
- Identify the most and least popular movies.
- Visualize trends using different chart types.
- Practice Exploratory Data Analysis (EDA) techniques.

---

## 🔍 Data Preprocessing

The following preprocessing steps were performed:

- Removed duplicate records
- Checked and handled missing values
- Converted data types
- Extracted release year from release date
- Split multiple genres into separate rows using `explode()`
- Reset index after transformations
